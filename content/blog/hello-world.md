# 歡迎來到我的部落格

這是使用 **Nuxt v4** 和 **@nuxt/content** 建立的現代化部落格網站。

## 功能特色

- ✅ Markdown 文章支援
- ✅ 語法高亮
- ✅ 響應式設計
- ✅ SEO 優化
- ✅ 暗色模式

## 程式碼範例

```javascript
// 這是一個 JavaScript 範例
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```
