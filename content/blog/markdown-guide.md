# Markdown 語法指南

Markdown 是一種輕量級標記語言，讓你可以使用純文字格式編寫文件。

## 基礎語法

### 標題

```markdown
# H1 標題
## H2 標題
### H3 標題
```

### 文字格式

- **粗體文字**
- *斜體文字*
- `行內程式碼`

### 清單

#### 無序清單

- 項目 1
- 項目 2
- 項目 3

#### 有序清單

1. 第一項
2. 第二項
3. 第三項

```javascript
function hello() {
  console.log('Hello, World!');
}
```

### 表格

| 功能 | 支援 | 說明 |
|------|------|------|
| 標題 | ✅ | H1-H6 |
| 清單 | ✅ | 有序/無序 |
| 程式碼 | ✅ | 語法高亮 |

這些是 Markdown 的基本語法，掌握後就能高效地撰寫文件了！
